Tema 4 - Animation

På dette tema skulle vi lave vores eget spil ved brug af html, css og javascript samt illustrator til at designe selve spillet og dets figurelementer. Hertil lærte vi at lave animationer og arbejde med javascript samt kendskab til illustrator og dets forskellige værktøjer. Derudover arbejdede vi med skitser og papirprototype under idefasen for at få inspiration til vores spil ift. hvad spillet skulle handle om samt spillets design. Yderligere udarbejdede vi et aktivitetsdiagram og et state machine diagram for få overblik over hvordan spillet skulle opgaves. For at forbedre spillet udarbejde jeg et spørgeskema.
